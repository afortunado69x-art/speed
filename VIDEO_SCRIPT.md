# GrimRead — Demo Video Script
## CodeCanyon / Envato Market · Duration: ~55 seconds

---

## PRE-PRODUCTION NOTES

**Tools needed:**
- Screen recorder: Android Studio Emulator (Pixel 7 Pro) or physical device
- Video editor: CapCut / DaVinci Resolve / Adobe Premiere
- Music: Dark ambient, ~55 BPM — "Dark Medieval" on Pixabay (royalty-free)
- Font overlay: Cinzel Decorative (matches app)
- Color grade: Deep contrast, crush blacks, slight desaturation + red channel boost

**Resolution:** 1080×1920 (portrait 9:16) for Envato preview
**Export:** H.264, 8 Mbps, AAC audio

---

## SHOT LIST & TIMING

---

### [0:00 – 0:04] COLD OPEN · SPLASH SCREEN

**Screen:** App launching — animated gothic splash
- Gothic cross fades in with blood-red glow
- "GrimRead" title slides up
- Gold loading dots pulse

**Camera:** Static, full screen capture
**Text overlay:** None — let the animation speak
**Audio:** Music fades in, low rumble + single deep bell toll

---

### [0:04 – 0:10] LIBRARY SCREEN

**Screen:** Library with wooden shelves and colored book spines
**Action sequence:**
1. Show full library with 2 loaded shelves
2. Slow scroll down to reveal second shelf
3. Tap "✚ Add Tome" button
4. File picker opens — select "master_i_margarita.fb2"
5. New book appears on shelf with slide-in animation

**Text overlay** (bottom, Cinzel font, gold):
```
7 formats: TXT · FB2 · EPUB · PDF · DOCX · RTF · MOBI
```

**Audio:** Soft page-turn sound on book import

---

### [0:10 – 0:16] MODE PICKER

**Screen:** Tap a book → mode picker bottom sheet rises
**Action sequence:**
1. Tap crimson book "Мастер и Маргарита"
2. Bottom sheet slides up showing RSVP ⚡ and Scroll 📜 cards
3. Hover moment (hold 1 second for viewer to read)
4. Tap ⚡ RSVP card

**Text overlay:**
```
Two reading modes — choose your ritual
```

---

### [0:16 – 0:28] RSVP READER — THE HERO SHOT

**Screen:** RSVP reader — this is the money shot, spend time here
**Action sequence:**
1. Open on word "Воланд" displayed — ORP letter glowing crimson
2. Tap play — words begin flowing at 250 WPM
3. Show 4–5 words cycling: Москва → шёл → по → узким → улочкам
4. Drag speed slider up: 250 → 400 → 600 WPM
5. Words blurring faster — gothic intensity effect
6. Tap pause
7. Tap chapter strip → jump to "Chapter V"
8. Word display jumps, new chapter word appears

**Text overlay** (appears at 0:18):
```
ORP Technology — eyes lock instantly
```
**Text overlay** (appears at 0:22, slides out):
```
50 – 1500 WPM · Punctuation pauses · Auto-save
```

**Audio:** Subtle whoosh on each word change (very quiet), music intensifies slightly

---

### [0:28 – 0:36] SCROLL READER + BOOKMARKS

**Screen:** Switch to scroll mode
**Action sequence:**
1. Tap "📜 Scroll" button in top-right of RSVP reader
2. Smooth transition to scroll reader — gothic parchment text
3. Scroll slowly through text showing chapter title with gold divider
4. Long-press a paragraph → bookmark ribbon appears with toast "✦ Bookmark added"
5. Tap bookmark icon in bottom bar
6. Bookmarks panel slides in from right showing 3 bookmarks with previews
7. Tap one bookmark → screen scrolls to position

**Text overlay** (at 0:29):
```
Scroll Mode — classical parchment reading
```
**Text overlay** (at 0:33):
```
Long-press to bookmark · Add notes
```

---

### [0:36 – 0:43] SETTINGS + LANGUAGES

**Screen:** Settings (Rites) screen
**Action sequence:**
1. Tap bottom nav → Rites screen
2. Show 8-language grid: RU · EN · DE · ES · PT · IT · FR · UA
3. Tap "EN" flag — UI instantly switches to English
4. Tap speed setting → speed slider sheet opens
5. Drag font size slider: text preview updates live
6. Cycle font family button: Gothic → Classical → Serif

**Text overlay:**
```
8 languages · Live font preview · Fully configurable
```

---

### [0:43 – 0:49] STATISTICS

**Screen:** Annals (Statistics) screen
**Action sequence:**
1. Tap "☽ Annals" in bottom nav
2. Stat stones animate in (staggered: WPM → Words → Books → Streak)
3. Scroll down to session history — show 3 recent sessions with WPM values
4. Numbers highlighted in gold

**Text overlay:**
```
Track your reading growth · Session history
```

**Screen:** Return to RSVP reader, one more beautiful word cycle
**Action sequence:**
1. Cross-fade back to reader screen

GrimRead
